Core apps & services

The Microsoft 365 apps we set up, secure, and support.

Collaboration & communicationIdentity & securityProductivity & AI

Microsoft Teams

Chat, meetings & calling

Chat, video meetings, and file collaboration included with every Microsoft 365 business and enterprise plan. We set up channel structure to match how your teams actually work, configure external guest access safely, and tune meeting policies so recordings and permissions land where compliance expects them.

CoversChannel setup, external guest access, Teams Phone add-on, meeting policies
  • Included with most M365 plans
  • Teams Phone available as add-on
  • Guest and external access configuration
  • Meeting recording and retention policy setup

Exchange Online

Hosted business email

Hosted mailboxes on your own domain, with spam filtering, mobile sync, and shared mailbox support. We handle MX and DNS record setup so mail routes correctly from day one, and run staged migrations from Gmail or on-prem Exchange so inboxes and calendars move over without downtime.

CoversMailbox provisioning, domain and DNS setup, shared/resource mailboxes, migration from legacy email
  • Custom domain email
  • Migration from Gmail or on-prem Exchange
  • Mailbox sizing per plan
  • SPF, DKIM, and DMARC configured for deliverability
  • Shared and resource mailbox setup for teams and equipment

SharePoint

Intranet & document libraries

Team sites, document libraries, and permission-managed intranets built on SharePoint Online. We design the site structure around your actual departments and projects, set retention and permission policies before content piles up, and connect it to Teams and OneDrive so files aren't duplicated across three places.

CoversSite structure, document libraries, permission groups, retention policies
  • Team and communication sites
  • Version history and co-authoring
  • Integrates with Teams and OneDrive
  • Permission group design to prevent oversharing
  • Retention labels and record declaration for compliance

OneDrive

Cloud file storage

Per-user cloud storage with file sync, sharing controls, and offline access across devices. We roll out the sync client fleet-wide, set external sharing defaults so files aren't shared more broadly than intended, and configure known-folder redirection so desktop, documents, and pictures back up automatically.

CoversStorage allocation per plan, sharing and link permissions, sync client rollout
  • 1TB+ storage depending on plan
  • Device sync configuration
  • External sharing controls
  • Known-folder redirection for automatic backup
  • Files On-Demand for lightweight local storage use

Microsoft Intune

Device management

Cloud-based device and app management for company-owned and BYOD endpoints across Windows, iOS, and Android. We build enrollment profiles once and apply them consistently across the fleet, set compliance policies that actually match your risk tolerance, and separate personal and corporate data on BYOD devices with app protection policies.

Available versionsIntune Plan 1, Plan 2, included in Microsoft 365 E3/E5
  • Device enrollment and policy
  • App protection policies
  • Compliance reporting
  • BYOD data separation for personal devices

Microsoft Defender

Endpoint protection

Endpoint, identity, and cloud threat protection built into the Microsoft 365 security stack. We enable the Defender components your plan includes, tune detection policies to cut false positives without missing real threats, and set up alert routing so incidents reach your team instead of sitting unread in a console.

Available versionsDefender for Endpoint, Defender for Office 365, Defender for Cloud
  • Endpoint policy configuration
  • Threat and alert monitoring
  • Integration with Microsoft 365
  • Alert routing and escalation setup

Microsoft Copilot

AI assistant

AI assistant embedded across Word, Excel, Outlook, and Teams for drafting, summarizing, and analysis. We review what data Copilot can see against your existing SharePoint and Teams permissions before rollout, so it doesn't surface content that was never meant to be that discoverable, then plan licensing and adoption in phases.

Available versionsCopilot for Microsoft 365, Copilot Chat
  • Licensing and rollout planning
  • Data and permissions review
  • Adoption and usage guidance
  • Oversharing risk review before go-live
  • Custom agent guidance with Copilot Studio

Power BI

Business intelligence

Dashboards and reporting connected to your business data, from simple visuals to enterprise-wide analytics. We connect it to your actual data sources, build the first set of dashboards with your team so the metrics match what people already track, and set refresh schedules so reports don't go stale.

CoversPower BI Pro (included in E5), Power BI Premium, dataset and dashboard setup
  • Included in Microsoft 365 E5
  • Dashboard and report design
  • Data source connections
  • Scheduled data refresh configuration
  • Row-level security for data access control

Word, Excel & PowerPoint

Core Office apps

The desktop, web, and mobile apps your team actually writes, models, and presents in every day. We handle deployment and activation across the fleet, set default save locations to OneDrive/SharePoint instead of local drives, and configure AutoSave and co-authoring so version conflicts stop happening.

CoversDesktop/web/mobile rollout, default save locations, co-authoring setup, template libraries
  • Included with most M365 plans
  • Desktop apps on Business Standard/Premium and E3/E5
  • Shared template and branding setup
  • AutoSave and version history configuration

Outlook

Email & calendar client

The desktop, web, and mobile client that sits on top of Exchange Online for mail, calendar, and contacts. We configure shared calendars and resource booking, set up mobile device profiles for company and BYOD phones, and tune focused inbox and rules so important mail doesn't get buried.

CoversMobile profile setup, shared calendars, resource booking, rules and focused inbox tuning
  • Desktop, web, and mobile access
  • Shared and resource calendar setup
  • Mobile device app configuration
  • Cross-device rule and signature sync
  • Add-in management for organization-wide tools

Microsoft Entra ID

Identity & access management

The identity backbone behind every Microsoft 365 sign-in, conditional access rule, and single sign-on connection. We configure multi-factor authentication and conditional access policies that match your risk profile, connect single sign-on for the other business apps you already use, and clean up stale guest accounts before they become a liability.

Available versionsEntra ID Free, Premium P1, Premium P2 (included in Business Premium and E3/E5)
  • Multi-factor authentication rollout
  • Conditional access policy design
  • Single sign-on for third-party apps
  • Guest access review and cleanup
  • Entra ID Protection risk-based sign-in policies

Microsoft Purview

Data governance & compliance

Data loss prevention, information protection, and eDiscovery across Exchange, SharePoint, Teams, and OneDrive. We set sensitivity labels and DLP policies around the data types that actually matter to your business, configure retention so legal holds and audits aren't a scramble, and keep policies from blocking everyday work.

Available versionsPurview DLP, Information Protection, eDiscovery Standard/Premium (varies by plan)
  • Sensitivity label and DLP policy design
  • Retention and legal hold configuration
  • eDiscovery case setup
  • Compliance reporting for auditors
  • Data map setup for cross-service data discovery
